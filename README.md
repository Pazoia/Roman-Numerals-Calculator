# Roman Numerals Calculator

# Client requirements

Build a calculator that takes roman numerals, and return the results also in roman numerals.

|  Input  | Output Lower than 10 |
| :-----: | :------------------: |
|  I + I  |          II          |
| II \* V |          X           |
|  V - I  |          IV          |
| IV / II |          II          |

|   Input    | Output betwen 11 and 99 |
| :--------: | :---------------------: |
|   X + IV   |           XIV           |
|  XX + XXV  |           XLV           |
| III \* IX  |          XXVII          |
|  L - XXX   |           XX            |
| LXXV / III |           XXV           |

# Technologies

- Python 3.8
- Pytest 5.4
